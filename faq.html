Frequently Asked Questions

お客様からよくいただくご質問をまとめました。
解決しない場合はお気軽にお問い合わせください。

🍙
メニューについて
Q
おにぎりの具材はカスタマイズできますか？
+
はい、一部のメニューは具材・海苔の有無・ご飯の量などをカスタマイズいただけます。店頭でスタッフにお気軽にお申し付けください。なお、具材の変更は在庫状況により承れない場合もございます。
Q
季節限定メニューはどのくらいの期間提供されますか？
+
季節限定メニューは概ね2〜3ヶ月ごとに入れ替わります。現在の季節限定メニューの情報はInstagramや店頭でお知らせしています。売り切れ次第終了となりますのでお早めにどうぞ。
Q
ベジタリアン・ビーガン向けのメニューはありますか？
+
はい、ベジタリアン対応のメニューをご用意しています。メニューに「🌿 ベジ」のマークが付いているものが該当します。ビーガン対応については店舗スタッフにご確認ください。
Q
おにぎりのカロリー・栄養成分を教えてください。
+
主要メニューのカロリー・栄養成分表は店頭のメニューボードにてご確認いただけます。詳細な情報が必要な場合は店舗スタッフまたはお問い合わせフォームよりご連絡ください。
🌿
アレルギー・食材について
Q
アレルギー対応はしてもらえますか？
+
はい、対応可能なアレルギーがございます。ご注文の際に店舗スタッフにアレルギーについてお申し出ください。ただし、当店のキッチンでは卵・小麦・大豆・魚介類等のアレルゲンを含む食材を使用しており、完全に交差汚染を防ぐことは難しい場合があります。重篤なアレルギーをお持ちの方はアレルギー情報ページをご確認の上、ご相談ください。
Q
グルテンフリーのメニューはありますか？
+
白米を使ったおにぎりはグルテンを含まない食材が多いですが、醤油・タレ類にはグルテンが含まれる場合があります。グルテンフリー対応が必要な方は店舗スタッフにご相談ください。「GF」マークのついたメニューも一部ございます。
📍
店舗・営業時間について
Q
営業時間を教えてください。
+
店舗によって営業時間が異なります。各店舗の営業時間は店舗一覧ページをご確認ください。祝日・特別営業日については各店舗のSNSでもお知らせします。
Q
予約は必要ですか？
+
通常のご来店は予約不要です。ただし、ケータリングやグループでのご利用（10名以上）の場合は事前にご予約をお願いしています。お問い合わせフォームからご連絡ください。
Q
駐車場はありますか？
+
ワイキキ本店・アラモアナ店には専用駐車場はございません。近隣の公共駐車場をご利用ください。カイルア店・カポレイ店・マウイ店には無料駐車場が隣接しています。詳しくは店舗情報をご確認ください。
🛵
注文・デリバリーについて
Q
デリバリーはできますか？
+
はい、DoorDash・Uber Eatsを通じてホノルル市内へのデリバリーを承っています。一部の店舗は対象外となる場合がありますので、アプリでご確認ください。
Q
オンラインショップで購入できますか？
+
はい、冷凍おにぎりや調味料・ギフトセットをオンラインショップにてご購入いただけます。ハワイ全域への配送に加え、米国本土への発送も対応しています。詳しくはオンラインショップをご覧ください。
Q
支払い方法を教えてください。
+
現金・クレジットカード（Visa / Mastercard / American Express）・Apple Pay・Google Pay・各種電子マネーがご利用いただけます。なお、空港店では現金のみとなっております。
